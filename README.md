## 🚀 Tecnologias Utilizadas

Esse projeto foi desenvolvido com as seguintes tecnologias:

- **HTML5**: Estrutura semântica e organização da página.
- **CSS3**: Estilização e layout responsivo.
- **Flexbox**: Alinhamento e distribuição de elementos na página.
- **Grid**: Estruturação da página com grids para organização dos componentes.

## 💻 Funcionalidades

- Layout voltado para dispositivos **desktop**.
- Exibição de notícias com título, imagem e descrição.
- Estilo moderno, com foco em uma interface limpa e fácil de ler.
